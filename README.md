# ChatBot

A cybersecurity chatbot built in C# that teaches users about staying safe online.

---

## Description

Chatbot is a console application that responds to user input and provides information on cybersecurity topics like safe browsing, phishing and password safety. The bot greets the user by name and uses a numbered menu to make it easy to navigate.

---

## Setup

1. Open Visual Studio Code
2. Open the project folder
3. Open the terminal in VS Code by pressing *Ctrl + `*
4. Type this command and press Enter to run the program:

```
dotnet run
```

---

## How To Use

When the program starts you will hear a voice greeting and see the menu. Type a number and press Enter to get a response.

| Type this  What it does |
|-------------------------|
| `1` | Shows the help menu 
| `2` | Explains the bots purpose 
| `3` | Ask the bot how it is doing 
| `4` | Safe browsing tips 
| `5` | Phishing warning signs 
| `6` | Password safety tips 
| `0` | Exit the chatbot 

---

## Features

- ASCII art logo on startup
- Voice greeting when the program launches
- Asks for your name and uses it throughout
- Numbered menu for easy navigation
- Colour coded text output
- Input validation for empty or invalid input
- Topics covered: safe browsing, phishing, password safety
